# Proyecto Sprint 7

## Aplicación web de análisis de vehículos usados

Esta aplicación fue desarrollada con Streamlit, Pandas y Plotly.

Permite explorar un conjunto de datos de anuncios de venta de vehículos usados mediante visualizaciones interactivas.

## Funcionalidades

- Visualización de tabla de datos
- Histograma de precios
- Gráfico de dispersión
- Aplicación web desplegada en Render

## Tecnologías utilizadas

- Python
- Streamlit
- Pandas
- Plotly
- Render
- GitHub
